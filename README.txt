Se trata de un programa codificado en php: 
Está conformado por dos archivos, el index.php, en el mismo tenemos un formulario el cual recoge los datos de una persona para que se registre en un sistema. 
Por otro lado, tenemos un archivo php, procesar.php, al cual llegan los datos por Post, el mismo codifica los datos, genera un array y, los prepara para ser enviado por ejemplo a un modelo o a una base de datos. 
Para que el proyecto se ejecute, es necesario tener instalado en tu computadora siguiente: 
Php version mínima 5.5
Un servidor, puede ser apache.
Nota: estos paquetes se instalan de manera automática si descargar xammp (https://www.apachefriends.org/es/index.html). 
Para ejecutar el programa en windows, bajo xammp, debes descargar todos los ficheros, luego en tu computadora lo colocas en la carpeta c/xampp/htdocs/nombreProyecto. 
Abres el panel de control de xampp, inicias el módulo apache. 
Luego ejecutas un navegador y tipeas localhost/nombreProyecto. Con ello lograras la ejecución del programa.
